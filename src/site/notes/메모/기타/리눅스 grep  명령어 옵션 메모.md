- -c : 일치하는 행의 수 출력
- -i : 대소문자 구별하지 않음
- -v : 일치하는 않는 행만 출력
- -n : 해당 문자열이 포함된 행의 번호를 함께 출력
- -l : 패턴이 포함된 파일의 이름 출력
- -H  : 패턴이 포함된 파일의 이름과 함께 내용 출력
- -w : 단어와 일치하는 행만 출력
- -x : 라인과 일치하는 행만 출력
- -r : 하위 디렉토리를 포함한 모든 파일에서 검색
- -m 숫자 : 최대로 표시될 수 있는 결과 제한
- -E : 찾을 패턴을 정규표현식으로 검색
- -F : 찾을 패턴을 문자열로 검색
